# Portfólio Dev

Projeto desenvolvido como trabalho final para a prova do curso do SENAC.

O objetivo do projeto foi criar um portfólio moderno e responsivo utilizando tecnologias web fundamentais, aplicando conceitos de estruturação, estilização e interatividade.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 📌 Funcionalidades

- Layout responsivo
- Seção de projetos
- Área de contato
- Animações e interações
- Design moderno e minimalista

## 🎯 Objetivo

Demonstrar conhecimentos em desenvolvimento web frontend, organização de projeto e criação de interfaces modernas.
